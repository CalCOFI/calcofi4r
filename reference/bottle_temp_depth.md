# Bottle data of temperature with depth (m)

Extended CalCOFI station bottle cast data with temperature (Celsius) as
example data frame for visualization functions. Data were filtered to
casts with a minimum of 50 depth readings.

## Usage

``` r
bottle_temp_depth
```

## Format

A data frame with 1,077 rows and 3 variables

- cast_count:

  unique identifier for bottle cast

- depth_m:

  depth below the surface in meters

- v:

  variable, in this case temperature (Celsius)
