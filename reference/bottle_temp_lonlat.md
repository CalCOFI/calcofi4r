# Bottle data of temperature in space (latitude, longitude)

Extended CalCOFI station bottle cast with temperature (Celsius) as
example data frame for visualization functions.

## Usage

``` r
bottle_temp_lonlat
```

## Format

A data frame with 9,865 rows and 3 variables

- lon:

  longitude

- lat:

  latitude

- v:

  variable, in this case temperature (Celsius)

## Source

<https://calcofi.org/sampling-info/station-positions/>
<https://calcofi.org/data/oceanographic-data/bottle-database/>
