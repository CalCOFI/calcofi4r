# Bottle data in space and time

CTD bottle cast data as example data frame for visualization functions.

## Usage

``` r
cc_bottle
```

## Format

A data frame (851,493: rows x columns) with variables:

- lon:

  longitude

- lat:

  latitude

- date:

  date

- quarter:

  quarter

- depth_m:

  depth below the surface in meters

- sta_dpos:

  difference in position, from 5 (nearshore), 10 (offshore) to 20
  (outside 113 station extended area)

- t_degc:

  temperature (Celsius)

- salinity:

  salinity (TODO: units)

- o2sat:

  oxygen saturation (TODO: units)
