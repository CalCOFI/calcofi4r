# List tables in CalCOFI database

Lists all available tables in a CalCOFI database release.

## Usage

``` r
cc_list_tables(version = "latest")
```

## Arguments

- version:

  Database version (default: "latest")

## Value

Character vector of table names

## Examples

``` r
if (FALSE) { # \dontrun{
cc_list_tables()
} # }
```
