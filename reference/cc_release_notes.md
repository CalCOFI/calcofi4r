# View CalCOFI database release notes

Displays the release notes for a specific database version.

## Usage

``` r
cc_release_notes(version = "latest")
```

## Arguments

- version:

  Version string or "latest" (default)

## Value

Character string with release notes

## Examples

``` r
if (FALSE) { # \dontrun{
cc_release_notes("v2026.02")
} # }
```
