# Update the mermaid library in the package

Update the mermaid library in the package

## Usage

``` r
updateMermaid(version = "")
```

## Arguments

- version:

  A [character](https://rdrr.io/r/base/character.html) of the desired
  version (the latest by default)

## Value

See value returned by
[download.file](https://rdrr.io/r/utils/download.file.html)
